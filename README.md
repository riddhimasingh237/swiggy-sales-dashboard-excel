# swiggy-sales-dashboard-excel
Advanced Excel data analytics project focused on food delivery insights, sales trends, restaurant performance, and customer ratings

# Project Overview
This project is an Advanced Excel Dashboard built to analyze Swiggy food delivery data.

The dashboard provides insights into:
Total Sales
Average Ratings
Order Trends
Restaurant Performance
Monthly Revenue Analysis

# Business Insights & KPIs
This dashboard helps analyze operational and sales performance using key business metrics such as:
* Revenue and Order Tracking
* Customer Rating Trends
* Restaurant Performance Evaluation
* Category-wise Sales Contribution
* Monthly Growth Analysis
* Peak Demand Identification
* Order Distribution Insights
* Customer Preference Analysis

# Key Solutions
* Identified top-performing restaurants contributing maximum revenue
* Tracked monthly sales trends to monitor business growth
* Analyzed customer ratings to improve service quality
* Detected most popular food categories based on order volume
* Helped identify peak ordering periods for better resource planning
* Compared restaurant performance using KPIs and visual analytics
